Everything you need to know about the FeynML platform.

FeynML uses Kolmogorov-Smirnov (KS) tests for continuous features and Chi-square tests for categorical features. We also compute the Population Stability Index (PSI) to provide a single drift severity score per feature.

ECE measures the difference between a model's predicted confidence and its actual accuracy. A perfectly calibrated model with 90% confidence should be correct 90% of the time. FeynML visualizes this using Reliability Diagrams.
